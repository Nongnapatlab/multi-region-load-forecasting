# คู่มืออ่านผลลัพธ์ (Output Reference Guide)

คู่มือนี้อธิบายว่าแต่ละไฟล์ output คืออะไร แต่ละคอลัมน์มีความหมายว่าอย่างไร และจะดู/ตรวจสอบค่าผิดปกติยังไง สำหรับใช้ตรวจสอบความถูกต้องของผลลัพธ์ด้วยตนเอง

---

## 1. `output/predictions/all_zones_predictions.csv`
ผลพยากรณ์ของการรันล่าสุด (ทุกภาค ทุก 30 นาที, 288 แถวต่อภาค = 6 วัน)

| คอลัมน์ | ความหมาย |
| :--- | :--- |
| **`for_date`** | วันที่และเวลาของจุดข้อมูล (ทุก 30 นาที) |
| **`actual`** | ค่าความต้องการไฟฟ้าจริง (ถ้า = 0 คือยังไม่เกิดจริง) |
| **`zone`** | รหัสภาค (CAC/MAC/NAC/NEC/SAC) |
| **`pred_xgb` / `pred_lgbm`** | ค่าพยากรณ์จาก XGBoost และ LightGBM |
| **`pred_lstm`** | ค่าพยากรณ์จาก LSTM (เป็น NaN ถ้าข้อมูลย้อนหลังไม่พอ) |
| **`pred_ensemble`** | ค่าเฉลี่ยถ่วงน้ำหนักของทุกโมเดล |

---

## 2. ไฟล์สรุป Predictions อื่นๆ
* **`today_predictions.csv`**: ผลพยากรณ์เฉพาะของวันนี้
* **`latest_predictions.csv`**: ผลพยากรณ์ข้อมูลล่าสุดที่มี
* **`future_predictions.csv`**: ผลพยากรณ์ล่วงหน้าที่ยังไม่เกิดจริง
* **`prediction_history.csv`**: ประวัติการรันสะสมทุกวัน

---

## 3. `output/metrics/all_zones_metrics.csv`
ตัวชี้วัดความแม่นยำของโมเดล

| คอลัมน์ | ความหมาย |
| :--- | :--- |
| **`MAPE`** | % ความคลาดเคลื่อนเฉลี่ย (ยิ่งน้อยยิ่งดี, <5% คือดีมาก) |
| **`MAE` / `RMSE`** | ความคลาดเคลื่อนเฉลี่ยเป็นหน่วย MW |
| **`BIAS`** | ทิศทางความผิดพลาด (+ พยากรณ์สูงไป, - พยากรณ์ต่ำไป) |

---

## 4. ไฟล์ Logs และ Dashboard
* **`feature_importance.csv`**: ปัจจัยที่มีผลต่อการพยากรณ์ (เช่น โหลดเมื่อวาน, อุณหภูมิ)
* **`distribution_shift_check.csv`**: ตรวจสอบความผิดปกติของข้อมูล (ดูค่า Z-score)
* **`diagnostics_summary.csv`**: เช็คคุณภาพข้อมูล (ค่าว่าง, ข้อมูลซ้ำ)
* **`decision_dashboard.csv`**: สรุปตัวเลข Plan และ Forecast สำหรับผู้บริหาร
* **`load_forecast_report.xlsx`**: ไฟล์ Excel รวมทุกอย่างไว้ในไฟล์เดียว
